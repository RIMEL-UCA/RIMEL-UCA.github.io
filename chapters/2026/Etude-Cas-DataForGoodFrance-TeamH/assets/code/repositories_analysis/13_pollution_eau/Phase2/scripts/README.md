# Http sources

Ce dossier contient 3 scripts 


`get_http_sources.py` : Indique dans un fichier csv `http_references.csv` le nombre de references http 

`get_http_sources_from_creation.py` : Indique dans un fichier csv `http_references_history.csv` le nombre de references http en fonction du temps (pour chaque commit)

Ce script doit être executer dans le repo git du projet étudié (utilisation des commandes git)

`http_history_to_vizu.py` : A partir du csv créer précédemment, créer une visualisation
